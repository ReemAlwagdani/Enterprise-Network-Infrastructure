Enterprise Network Infrastructure

Overview:
- Designed and implemented a multi-branch enterprise network using Cisco Packet Tracer. The project includes VLSM subnetting, router and switch configuration, WAN connectivity, DHCP, DNS, and web services.

Features :
- VLSM Subnet Design
- Router Configuration
- Switch Configuration
- WAN Connectivity
- Static IP Addressing
- DHCP Server Configuration
- DNS Server Configuration
- Web Server Deployment
- Network Connectivity Testing

Technologies Used:
- Cisco Packet Tracer
- IPv4 Addressing
- VLSM Subnetting
- Routing
- DHCP
- DNS
- Web Services

Project Files :
- Enterprise_Network_Infrastructure.pkt
- Phase1_VLSM_Design.pdf
- Phase2_Network_Implementation_Documentation.pdf

Learning Outcomes:
- Network Infrastructure Design
- IP Address Planning
- Router and Switch Configuration
- DHCP and DNS Deployment
- WAN Connectivity
- Network Troubleshooting
